# 🚀 Personal Portfolio - Sergio Rodríguez Geniz

Bienvenido a mi portfolio personal. Este repositorio contiene el código fuente de mi sitio web profesional, diseñado para mostrar mi trayectoria, habilidades y proyectos como Desarrollador de Aplicaciones Multiplataforma (D.A.M.).

## 👨‍💻 Sobre Mí
Soy **Sergio Rodríguez Geniz**, Técnico Superior en D.A.M. apasionado por construir soluciones eficientes y elegantes. Me especializo en el desarrollo backend con un fuerte enfoque en tecnologías como Python (Django, FastAPI) y Java (SpringBoot, Hibernate), sin descuidar la creación de interfaces de usuario intuitivas y modernas.

## 🛠️ Tecnologías Utilizadas

### Core
*   **HTML5** & **CSS3**: Estructura semántica y diseño personalizado sin frameworks externos (Vanilla CSS).
*   **JavaScript (ES6+)**: Interactividad y lógica personalizada.

### Características del Diseño
*   **Bento Grid Layout**: Una distribución moderna y limpia para las secciones de información.
*   **Glassmorphism & Animaciones**: Uso de transparencias, gradientes y animaciones suaves al hacer scroll (Intersection Observer API).
*   **Efectos Dinámicos**: Parallax suave en elementos de fondo reactivo al movimiento del ratón.
*   **Responsive Design**: Totalmente adaptado a cualquier dispositivo (Móvil, Tablet, Escritorio).

## 🚀 Funcionalidades Clave
- 📱 **Diseño Adaptativo**: Experiencia optimizada para todos los tamaños de pantalla.
- ⏬ **Descarga de CV**: Integración para visualización y descarga directa de currículum en PDF.
- 🎨 **Animaciones Modernas**: Revelación progresiva de contenido al navegar.
- 📁 **Arquitectura Limpia**: Separación clara de estilos, scripts y estructura de archivos.

## 📂 Estructura del Proyecto
```text
.
├── docs/           # Documentos del proyecto (CV)
├── icons/          # Recursos gráficos e iconos
├── img/            # Imágenes del proyecto
├── languages/      # Idiomas del proyecto
├── scripts/        # Lógica de interactividad (script.js)
├── style/          # Hojas de estilo (style.css)
├── index.html      # Punto de entrada principal
├── sitemap.xml     # Mapa del sitio web para la indexación de la página en Google
└── README.md       # Documentación del proyecto
```
